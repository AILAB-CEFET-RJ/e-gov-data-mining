# e-gov-data-mining
Projeto de mineração de dados em Notas Fiscais Eletrônicas (NFE)


## Requirements

Conda 4.11.0 or latest

## Install conda environment

In terminal, type:
```
conda env create -f config/environment.yml
```

## Activate environment
```
conda activate egov
```

## Deactivate environment
```
conda deactivate
```
